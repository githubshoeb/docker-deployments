Local superset Evironment Setup Steps For Docker
======================================================

1. Installing superset Locally:
----------------------------------------------
Go into the linux directory and Open command prompt or terminal at superset folder
```
export TAG=3.1.1
docker compose -f docker-compose-image-tag.yml up
docker compose ps
```

refrence:
```
https://superset.apache.org/docs/installation/docker-compose/
```
